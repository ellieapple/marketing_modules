## üöÄ Quick Start
**IMPORTANT:** Some scripts in this repository are intentionally non-functional for copyright/security purposes. You'll need to test each one to determine which are production-ready.

### Installation

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt` (if present)
3. Run each script individually to verify functionality

#### Site Setup (Run First)
```bash
python setup_site.py
```
Configures HTML pages and generates required assets. Run this before testing the landing page.
### Available Scripts

#### Email Automation
```bash
python email_automation.py
```
Automates email campaign deployment and tracking. Handles bulk sends with personalization.

#### Lead Scraper
```bash
python lead_scraper.py
```
Extracts qualified contact information from target websites. Includes validation and export features.

#### SEO Analyzer
```bash
python seo_analyzer.py
```
Comprehensive SEO audit tool. Analyzes page speed, keywords, backlinks, and mobile optimization.

#### Backup Generator
```bash
python backup_generator.py
```
Creates encrypted backups of website files with cloud upload capability.
